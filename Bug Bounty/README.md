# Bug Bounty Hunting

### What Is Bug Bounty?
A program where security researchers find and repott vulnerabilities in exchange for rewards.

### Relation to Pentesting
Bug bounty mimics real-world penetration testing but with publicly known programs and scopes.

### Popular Platforms
 - [HackerOne](https://hackerone.com/opportunities/all)
 - [Bugcrowd](https://bugcrowd.com/engagements?category=bug_bounty&page=1&sort_by=promoted&sort_direction=desc)
